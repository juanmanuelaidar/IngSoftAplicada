# Legacy root JHipster app

Esta carpeta conserva la aplicacion JHipster que antes estaba generada en la raiz del repositorio, incluyendo sus wrappers, configuraciones y devcontainer.

La fuente de verdad de la entrega es `../../backend/`. Los comandos del `Makefile`, el `Jenkinsfile`, Docker y la documentacion principal apuntan a `backend/`.

Se deja este contenido archivado para consulta historica y para evitar borrar codigo sin revisar. No deberia usarse para build, tests ni deploy de la entrega.
